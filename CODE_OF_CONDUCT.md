# Code Of Conduct

This project is for people learning and building together around open small language models.

Be kind, direct, and constructive. Assume good intent, but respect boundaries. Do not harass, threaten, demean, or exclude people based on identity, background, experience level, or technical choices.

Keep discussions focused on reproducible behavior, evidence, and useful model recipes. If a contribution has safety, licensing, or quality problems, explain the issue clearly and help the contributor find a better path.

Maintainers may remove comments, issues, pull requests, or contributors that make the project unsafe or hostile.
